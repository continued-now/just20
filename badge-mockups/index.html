My read

Use Pins as the main badge system.

Option B is the strongest direction now: it makes badges feel more earned and more collectible without getting intense. I would use the enamel-pin system for unlocks and badge details, keep a simplified sticker treatment for tiny grid states, and reserve patches for squad/social moments later.
